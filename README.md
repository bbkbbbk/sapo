# sapo

sapo, your friendly whale friend
he can help you explore the world of music even more
what he can do

- know your favorite tracks
- knwo your favorite artists
- random you a song
- create a personalized playlist just for you

### Developed with

- Go, Echo
- Spotify API
- LINE Messaging API
- LIFF + Firebase hosting
- DigitalOcean Paas
